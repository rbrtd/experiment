# Sys_ChartOfAccounts
Об'єкт представляє Список планів рахунків
## Структура об'єкта

| Field      | Title    | Title Ua         | Type    | Linked object | Key / Display | Required | Calculated | Lookup | Size / Scale | Remark |
| :--------- | :------- | :--------------- | :------ | :------------ | :-----------: | :------: | :--------- | :----- | :----------- | :----- |
| ID         | ID       | Унікальний номер | number  |               |      Key      |    x     |            |        |              |        |
| Name       | Name     | Найменування     | string  |               |    Display    |    x     |            |        | 255          |        |
| IsArchived | Archived | Архівний         | boolean |               |               |          |            |        |              |        |
## Операції

| Name | Privilege | Mandatory Fields | Remark                           |
| :--- | :-------- | :--------------- | :------------------------------- |
| List | 854       |                  | Отримання списку планів рахунків |
| Ins  | 854       |                  | Створення нового плану рахунків  |
| Mod  | 854       | ID               | Редагування плану рахунків       |
| Del  | 854       | ID               | Видалення плану рахунків         |
### Загальні перевірки для усіх операцій
- Наявність відповідної привілеї, у випадку відсутності повертається помилка **`403`**
- Наявність обов'язкових полів операції, у випадку відсутності повертається помилка **`409`**
- Наявність обов'язкових фільтрів операції, у випадку відсутності повертається помилка **`410`**

### List
- Без додаткових умов та перевірок

### Ins
* Перевіряється унікальність по полю `Name`. При порушенні повертається помилка **`1102`**

### Mod
* Перевіряється унікальність по полю `Name`. При порушенні повертається помилка **`1102`**

### Del
* Якщо існують записи [Sys_Accounts](Sys_ChartOfAccounts.md).`ChartOfAccountsID` = `ID`, то повертається помилка **`8000`**
